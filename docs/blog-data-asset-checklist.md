# Blog Data Asset Checklist

This document lists the most useful data-support opportunities for the 40 blog posts under `src/content/blog`.

Use it as a working checklist when preparing charts, comparison tables, maps, timelines, or sourced visuals that can strengthen article credibility.

## How To Use This Checklist

- `Priority A`: strongest candidates for data-backed charts or tables
- `Priority B`: useful if data is available, but not required for the first content pass
- `Priority C`: optional visual support, usually diagrams or simple comparison graphics rather than heavy datasets

Recommended output format for the site:

- static PNG or WebP charts exported into `public/images/blog/`
- one chart per article is usually enough
- use short chart titles that can also work as figure captions

## Recommended Source Types

When preparing data, prefer:

- official government statistics
- industry associations
- standards organizations
- customs or trade databases
- internal sourcing observations when clearly labeled as internal benchmarks

If exact public data is difficult to confirm, we can also use:

- comparison tables
- scoring frameworks
- process diagrams
- inspection checklists

These still strengthen the article even when hard numeric datasets are limited.
